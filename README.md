# Projeto E-lixo

Aplicacao web voltada ao tema de lixo eletronico, reciclagem, informacao ambiental e interacao com usuarios.

## Objetivo

Organizar informacoes e fluxos relacionados ao descarte correto de lixo eletronico, com paginas de apoio, cadastro, login, entregas e loja de pontos.

## Tecnologias

- HTML
- CSS
- JavaScript
- Firebase Hosting

## Como visualizar

Abra `index.html` em um navegador ou publique a aplicacao em um ambiente Firebase/GitHub Pages configurado.

## Estrutura

- `index.html`: entrada principal
- `pages/`: telas internas da aplicacao
- `firebase-init.js`: configuracao do Firebase
- `firebase.json`: configuracao de hospedagem
- `storage.rules`: regras de armazenamento versionadas

## Observacao de seguranca

A configuracao Firebase de frontend pode aparecer em projetos web, mas as regras do Firebase precisam estar bem protegidas. Revise Firestore, Authentication e Storage no console do Firebase antes de usar dados reais.
