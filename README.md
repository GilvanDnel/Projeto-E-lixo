# Projeto E-lixo

Aplicacao web voltada ao tema de lixo eletronico, reciclagem, informacao ambiental e interacao com usuarios.

## Acesse

https://gilvandnel.github.io/Projeto-E-lixo/

## Objetivo

Organizar informacoes e fluxos relacionados ao descarte correto de lixo eletronico, com paginas de apoio, cadastro, login, entregas e loja de pontos.

## Tecnologias

- HTML
- CSS
- JavaScript
- Firebase Hosting/configuracao frontend
- GitHub Pages

## Estrutura

- `index.html`: entrada principal
- `pages/`: telas internas da aplicacao
- `firebase-init.js`: configuracao Firebase do frontend
- `firebase.json`: configuracao de hospedagem
- `storage.rules`: regras de armazenamento versionadas

## Observacao de seguranca

A configuracao Firebase de frontend pode aparecer em projetos web, mas as regras do Firebase precisam estar bem protegidas. Antes de usar dados reais, revise Firestore, Authentication e Storage no console do Firebase.

## Possiveis melhorias

- Documentar fluxo principal do usuario.
- Adicionar prints das telas.
- Revisar regras do Firebase antes de uso em producao.
